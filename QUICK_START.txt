## â“ FAQ

**Q: Do I need both terminals?**  
A: Yes. Backend = data API, Frontend = user interface. Both required.

**Q: Can I use Python 3.12 instead of 3.13?**  
A: Yes, any 3.10+ works. 3.13 is just tested/recommended.

**Q: Is internet required?**  
A: Only for initial setup (pip install). Can run offline after.

**Q: Can I access from another computer?**  
A: Yes if deployed to cloud. Local setup only accessible on same machine.

**Q: How do I backup my account?**  
A: Docker container has `users.db` backed up. Or use `git` for version control.

**Q: Can I customize the theme?**  
A: Yes! Edit CSS in `frontend/app.py` and individual page files.

**Q: How do I reset my password?**  
A: Delete `backend/users.db` and restart. Users recreate accounts on next login.
